# Bem Vindo ao BIORes!

**BIORes** é um projeto para estudo particular de ferramentas de **web design** e **web dev** (além das matérias da faculdade :D).

É um site que, *por ora*, apenas comporta resumos prontos das matérias, mas há a pretenção de implementar novas funcionalidades, como:

* Criação de conta e perfil;
* Salvar resumos favoritos;
* Salvar trechos de destaque.

É um projeto inicial (como um teste)! Após sua conclusão, é planejada a criação de um webapp de criação, formatação e gerenciamento de resumos.
